# BuildwithTreez Brand Guide

## 🎨 Brand Identity

**Brand Name:** BuildwithTreez  
**Tagline:** Immersive 3D visuals, intuitive user interfaces, and dynamic web applications  
**Personality:** Innovative, Professional, Creative, Technical Excellence

---

## 🌈 Color Palette

### Primary Colors
```css
--treez-purple: #915eff;    /* Primary brand color - innovation & creativity */
--treez-cyan: #00d4ff;      /* Secondary - technology & trust */
--treez-pink: #ff6b9d;      /* Accent - energy & passion */
```

### Supporting Colors
```css
--deep-space: #0a0a0f;      /* Primary background */
--cosmic-gray: #1a1a2e;     /* Secondary background / cards */
--soft-white: #f8f8f8;      /* Primary text */
--muted-gray: #b0b0b0;      /* Secondary text */
```

### Gradient Combinations
```css
/* Primary Gradient */
background: linear-gradient(135deg, #915eff 0%, #ff6b9d 100%);

/* Tech Gradient */
background: linear-gradient(135deg, #00d4ff 0%, #915eff 100%);

/* Energy Gradient */
background: linear-gradient(135deg, #ff6b9d 0%, #915eff 50%, #00d4ff 100%);
```

---

## 🎯 Usage Guidelines

### Primary Purple (#915eff)
- **Use for:** Primary CTAs, brand elements, key highlights
- **Represents:** Innovation, creativity, premium quality
- **Pairs with:** Cyan, Pink, White

### Cyan (#00d4ff)
- **Use for:** Secondary CTAs, tech elements, links
- **Represents:** Technology, trust, clarity
- **Pairs with:** Purple, White, Deep Space

### Pink (#ff6b9d)
- **Use for:** Accents, hover states, energy elements
- **Represents:** Passion, energy, creativity
- **Pairs with:** Purple, Cyan, White

---

## 🎨 Sacred Geometry Theme

### Merkaba Colors
- **Outer Tetrahedron:** Bright Magenta (#ff6bff) - slightly lighter than brand purple
- **Inner Tetrahedron:** Bright Cyan (#00ffff) - pure cyan for contrast
- **Opacity:** 90% outer, 80% inner for depth

### Quantum Field
- **Particles:** Cyan (#00d4ff) - matches brand secondary
- **Glow Effect:** Bloom post-processing for ethereal feel

### Background Pattern
- **Flower of Life:** Gradient using all three brand colors
- **Opacity:** 7% for subtle presence
- **Blend Mode:** Screen for luminous effect

---

## 📐 Typography

### Headings
- **Font:** Bold, modern sans-serif
- **Color:** White (#ffffff) for maximum contrast
- **Effect:** Subtle glow on hover using brand colors

### Body Text
- **Primary:** Soft White (#f8f8f8)
- **Secondary:** Muted Gray (#b0b0b0)
- **Links:** Cyan (#00d4ff) with purple hover

---

## 🔘 Button Styles

### Primary CTA
```css
background: linear-gradient(to right, #915eff, #ff6b9d);
color: white;
hover: reverse gradient + scale(1.05)
shadow: glow effect with brand color
```

### Secondary CTA
```css
border: 2px solid #00d4ff;
color: #00d4ff;
hover: background #00d4ff/10 + scale(1.05)
shadow: cyan glow on hover
```

### Ghost Button
```css
border: 1px solid rgba(255, 255, 255, 0.2);
color: white;
hover: border-color #915eff
```

---

## ✨ Interactive Elements

### Hover Effects
- **Scale:** 1.05 (subtle lift)
- **Shadow:** Colored glow matching element
- **Transition:** 300ms ease-in-out
- **Gradient:** Reverse or shift on hover

### Focus States
- **Outline:** 2px solid brand color
- **Glow:** Soft shadow in brand color
- **Accessibility:** High contrast maintained

---

## 🎭 Brand Applications

### Hero Section
- **Background:** Deep Space with Flower of Life
- **3D Elements:** Merkaba (purple/cyan) + Quantum Field (cyan)
- **Text:** White primary, gray secondary
- **CTA:** Purple-to-pink gradient primary, cyan border secondary

### Cards & Sections
- **Background:** Cosmic Gray with subtle gradient
- **Border:** Subtle glow in brand colors
- **Hover:** Lift effect with colored shadow

### Navigation
- **Links:** White with cyan hover
- **Active:** Purple underline or highlight
- **Mobile:** Smooth transitions with brand colors

## 🚀 Consistency Checklist

- [ ] All CTAs use brand gradient or cyan border
- [ ] Hover states include scale + glow effects
- [ ] Sacred geometry uses brand colors
- [ ] Backgrounds maintain deep space theme
- [ ] Text maintains high contrast (WCAG AA+)
- [ ] Interactive elements have 300ms transitions
- [ ] Gradients flow in consistent direction (135deg)
- [ ] Shadows use brand colors, not gray

## 💡 Design Philosophy

**"Where Sacred Geometry Meets Modern Technology"**

The BuildwithTreez brand combines ancient sacred geometry patterns with cutting-edge web technology, creating a unique visual identity that represents both timeless design principles and innovative technical expertise.

**Key Principles:**
1. **Subtle but Impressive** - Effects enhance, never distract
2. **Professional Polish** - Enterprise-grade quality
3. **Accessible** - Beautiful for all users
4. **Consistent** - Brand recognition through repetition
5. **Innovative** - Push boundaries while maintaining usability
