## 📝 How to Use These Prompts

1. **Choose a concept** that resonates with you
2. **Copy the prompt** exactly as written
3. **Paste into AI tool** (Microsoft Designer, Recraft.ai, Ideogram.ai, etc.)
4. **Generate 4-8 variations** per prompt
5. **Try multiple prompts** - generate 20-30 total options
6. **Pick your top 5** for refinement in GIMP

**Recommended Tools:**
- **Microsoft Designer** (https://designer.microsoft.com) - FREE, best quality
- **Recraft.ai** (https://recraft.ai) - FREE, vector output
- **Ideogram.ai** (https://ideogram.ai) - FREE, great with text
- **Leonardo.ai** (https://leonardo.ai) - FREE tier, high control

## 🎨 Prompt Modifiers

**Add these to any prompt above for variations:**

### Make it Simpler
```
"Even more minimal. Remove all details except essential shapes. 
Maximum simplicity while maintaining recognition."
```

### Add More Detail
```
"Add subtle details: fine lines, small geometric patterns, 
or sacred geometry background elements."
```

### Change Style
```
"Same concept but in [choose style]:"
- "brutalist geometric style"
- "organic flowing style"  
- "futuristic sci-fi style"
- "retro 80s tech style"
- "sacred geometry mandala style"
```

### Adjust Colors
```
"Use only purple #915eff as single color, monochrome"
"Use only teal #00d4ff as single color, monochrome"
"Add pink accent #ff6b9d to the gradient"
"Black and white version only"
```

### Request Variations
```
"Generate 4 variations with different tree styles"
"Show both filled and outline versions"
"Create icon-only and icon-with-text versions"
```

## 📋 Generation Workflow

### Step 1: Bulk Generate (30-60 min)
- [ ] Pick 5-7 favorite prompts from above
- [ ] Generate 4-6 variations per prompt
- [ ] Save ALL results (name them by concept)
- [ ] Total target: 20-30 logo options

### Step 2: First Review (10 min)
- [ ] Eliminate obvious "no" options
- [ ] Look for 5-8 "maybe" options
- [ ] Identify 2-3 "strong yes" options

### Step 3: Refinement (15 min)
- [ ] Take "strong yes" options
- [ ] Generate variations with modifier prompts
- [ ] Try different colors on best designs
- [ ] Request both icon-only and full versions

### Step 4: Final Selection (10 min)
- [ ] Compare top 3-5 side by side
- [ ] Test at different sizes (zoom in/out)
- [ ] Imagine on website header
- [ ] Pick THE ONE (or top 2)

## 💡 Pro Tips

### For Best Results:

**✅ DO:**
- Generate MANY options before deciding
- Try multiple AI tools (each has strengths)
- Request both color and monochrome versions
- Ask for transparent backgrounds
- Save everything (even "failures" might inspire later)
- Test readability at 16px size

**❌ DON'T:**
- Settle on the first good option
- Use more than 3 colors in a logo
- Make it too detailed (won't scale down)
- Add text if AI struggles with it (add later in GIMP)
- Ignore alignment with portfolio aesthetic

### Testing Checklist:
- [ ] Readable at 16x16px?
- [ ] Works on light background?
- [ ] Works on dark background?
- [ ] Recognizable in 1 second?
- [ ] Unique and memorable?
- [ ] Represents brand values?
- [ ] Aligns with portfolio design?
- [ ] Professional enough for resume?

## 🔄 Iteration Examples

**If your first result is close but needs adjustment:**

### Too Complex
```
"Simplify this design. Remove 50% of details. 
Keep only the essential shapes and main concept."
```

### Too Simple
```
"Add subtle sacred geometry pattern in background. 
Maintain minimalist aesthetic but add one layer of detail."
```

### Wrong Colors
```
"Same design but change colors to: 
deep purple #915eff and bright cyan #00d4ff only. 
No other colors."
```

### Wrong Proportions
```
"Make the tree element larger and more prominent. 
Adjust spacing and balance."
```
